OBJECTIFS: Créér une application web dont le thème est basé sur le cirque

SUGGESTIONS Pour le contenu: 
- un tableau d'administration pour éditer les spectacles
- un système de géolocalisation
- un système de réservation de billets
- une galerie qui affiche les artistes wilders

User stories: 
- En tant utilisateur, je veux accéder à la page de réservation de billets
- En tant  utilisateur, je veux conxulter la liste des spectacles disponibles
- En tant administrateur, je veux modifier mes spectacles
- En tant administrateur, je veux accéder à ma base de données

Mise en place du projet wild Circus:

- je vais créer 2 dossiers:
 - server (back-end)
 - client (front-end)
le coté client contiendra tout le code react. Tandis que le coté server y contiendra le code nodejs et mysql.
